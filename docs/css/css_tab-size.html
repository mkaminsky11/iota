Because this technology's specification has not stabilized, check the compatibility table for the proper prefixes to use in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the spec changes.

Summary

The tab-size CSS property is used to customize the width of a tab (U+0009) character.

  • Initial value 8
  • Applies to block containers
  • Inherited yes
  • Media visual
  • Computed value the specified integer or an absolute length
  • Animatable no
  • Canonical order the unique non-ambiguous order defined by the formal grammar

Syntax

Formal syntax: | <length"><integer> | <length>

tab-size: 4
tab-size: 0
tab-size: 8

tab-size: inherit

Values

<integer>
The number of spaces in a tab. Must be positive.
<length>
The width of a tab. Must be positive.
inherit
Inherits from the parent element.
