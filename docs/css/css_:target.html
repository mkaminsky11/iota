:target

Summary

The :target pseudo-class represents the unique element, if any, with an id matching the fragment identifier of the URI of the document..

URIs with fragment identifiers link to a certain element within the document, known as the target element. For instance, here is a URI pointing to an anchor named section2:
http://example.com/folder/document.html#section2
The anchor can be any element with an id attribute, e.g. <h1 border: 1px dotted gray;">section2"> in our example. The target element h1 can be represented by the :target pseudo-class.

Note: The id attribute was new in HTML 4 (December 1997).  In old-style HTML <a name="foo"> is a target element.  The :target pseudo-class applies to those targets as well.
